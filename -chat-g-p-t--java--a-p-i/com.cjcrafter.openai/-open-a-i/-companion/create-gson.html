@JvmStatic
fun createGson(): Gson

Returns a Gson object that can be used to read/write .json files. This can be used to save requests/responses to a file, so you can keep a history of all API calls you've made.

This is especially important for ChatRequest, since users will expect you to save their conversations to be continued at later times.

If you want to add your own type adapters, use createGsonBuilder instead.

Return

Google gson serializer for json files.
