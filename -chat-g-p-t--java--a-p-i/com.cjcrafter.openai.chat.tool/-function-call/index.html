FunctionCall

data class FunctionCall(var name: String, var arguments: String)

Represents a function call by ChatGPT. When ChatGPT calls a function, you should be parsing the arguments, calling some method (getting current weather at a location, getting a stock price, modifying a database, etc.), and then replying to ChatGPT with the result.

ChatGPT may hallucinate, or make up, function calls. To handle hallucinations, we have provided tryParseArguments.
