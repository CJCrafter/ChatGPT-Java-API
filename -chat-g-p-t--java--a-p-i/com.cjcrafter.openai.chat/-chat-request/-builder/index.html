Builder

class Builder

Builder is a helper class to build a ChatRequest instance with a fluent API. It provides methods for setting the properties of the ChatRequest object. The build method returns a new ChatRequest instance with the specified properties.

Usage:

val chatRequest = ChatRequest.builder()
    .model("gpt-3.5-turbo")
    .messages(mutableListOf("Be as helpful as possible".toSystemMessage()))
    .temperature(0.7f)
    .build()
